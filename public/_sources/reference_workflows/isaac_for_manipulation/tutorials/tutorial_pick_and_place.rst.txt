=====================================
Multi-Object Pick and Place Tutorials
=====================================

Overview
--------

The Multi-Object Pick and Place tutorials demonstrate GPU-accelerated manipulation workflows 
that handle multiple objects.

.. figure:: :ir_lfs:`<resources/isaac_ros_docs/reference_workflows/isaac_for_manipulation/pick_and_place_sped_up.gif>`

   :alt: Multi object pick and place example.
