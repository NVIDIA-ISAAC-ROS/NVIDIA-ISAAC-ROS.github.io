Grounding DINO
==============

`Grounding DINO <https://github.com/IDEA-Research/GroundingDINO>`__ is an open-set object detection model able to detect arbitrary objects specified by natural language inputs.

To learn more about the model architecture and using Grounding DINO with the `TAO Toolkit <https://developer.nvidia.com/tao-toolkit>`__, see `the NGC model card <https://catalog.ngc.nvidia.com/orgs/nvidia/teams/tao/models/grounding_dino>`__.
