|repo_name|
===========

This meta-repository contains the packages required for the
`Nova Carter <https://robotics.segway.com/nova-carter/>`__
robots to control their motors / wheelbase, communicate with the
`Mission Dispatcher <https://github.com/nvidia-isaac/isaac_mission_dispatch>`__,
navigate with
`Nav2 <https://github.com/ros-planning/navigation2>`__, localize with
:doc:`Isaac ROS Visual SLAM </repositories_and_packages/isaac_ros_visual_slam/index>`,
and reconstruct 3D scenes with
:doc:`Isaac ROS Nvblox </repositories_and_packages/isaac_ros_nvblox/index>`.
