Supported Platforms
-------------------

This package is designed and tested to be compatible with ROS 2 Humble running on `Jetson <https://developer.nvidia.com/embedded-computing>`__ platforms. See the compatibility of `jetson_stats <https://rnext.it/jetson_stats/#compatibility>`__.

.. note::

   x86_64 system is not supported.
