Hawk Stereo Camera Examples
===========================

.. figure::
    :ir_lfs:`<resources/isaac_ros_docs/repositories_and_packages/isaac_perceptor/perceptor_in_zanker.gif>`
    :width: 800px
    :alt: Foxglove visualization of the Isaac Perceptor outputs
    :align: center

Nvblox can be run on the `Hawk stereo camera <https://leopardimaging.com/leopard-imaging-hawk-stereo-camera/>`__
as part of Isaac Perceptor, a camera-based 3D perception system for mobile robots that leverages
multiple Isaac ROS modules such as
:doc:`Isaac ROS Nova </repositories_and_packages/isaac_ros_nova/index>`,
:doc:`Isaac ROS Visual SLAM </repositories_and_packages/isaac_ros_visual_slam/index>`,
:doc:`Isaac ROS Depth Estimation </repositories_and_packages/isaac_ros_dnn_stereo_depth/index>`
and :doc:`Isaac ROS Nvblox </repositories_and_packages/isaac_ros_nvblox/index>`.

Refer to :doc:`Isaac Perceptor </reference_workflows/isaac_perceptor/index>` if you wish to know more.

To run nvblox as part of Isaac Perceptor see the Isaac Perceptor tutorials:

* :doc:`Tutorial: Recording and Playing Back Data for Isaac Perceptor </reference_workflows/isaac_perceptor/tutorials_on_devkit/tutorial_recording_and_playback>`
* :doc:`Tutorial: Isaac Perceptor on Nova Orin Developer Kit </reference_workflows/isaac_perceptor/tutorials_on_devkit/demo_perceptor>`
* :doc:`Tutorial: Isaac Perceptor on Nova Carter </reference_workflows/isaac_perceptor/tutorials_on_carter/demo_perceptor>`
