Platforms
---------

Nova supports any variant of the Nova sensor suite on a Jetson AGX Orin. There are also multiple
products that use Nova and have supported examples below. If you are not using one of the example
products, follow the :doc:`adapting Nova</getting_started/platforms/adapting_nova>` guide.
